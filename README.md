## Getting Started

### Installing

Clone the repo:

```
git clone git@github.com/StianL82/portfolio2
```

Install dependencies

```
npm install
```

## Run the project

Start the development server

```
npm run dev
```

### Running

Open with live server, or open in browser [Link to page](https://portfolio2-stianl.netlify.app/).
